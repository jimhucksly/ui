* использование в html

```html
<div style="background-color: var(--primary);"></div>
```

* использование в css

```css
.className {
  color: var(--primary);
}
```

* text color class names

** white--text
** primary--text
** secondary--text
** success--text
** grey--text
** warning--text
** error--text

```html
<span class="primary--text">Text</span>
```
