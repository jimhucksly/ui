```html
<ld-edit-text
  type="password"
  v-model="model"
  v-model:value="password"
  label="Password"
/>
```

model - здесь будет значение, соответствующее режиму отображения,
ля редактирования сюда нужно записать старое значение

password - здесь будет сам пароль, это значение необходимо использовать,
для сохранения на сервере
