По умолчанию подключается коллекция перевода на русский язык

```js
'node_modules/@dn-web/ui/dist/i18n/ru/ru-Ru.json'
```
Если необходимо использовать другую коллекцию

* подключаем подсистему перевода

```js
import ui, { i18n } from '@dn-web/ui';

appComponent.use(ui, { language: 'en' });
appComponent.use(i18n);
```

имеющуюся коллекцию перевода на английский язык можно посмотреть здесь
```js
'node_modules/@dn-web/ui/dist/i18n/en/en-En.json'
```
Если нет нужной библиотеки перевода, то можно создать свою

```js
const library = JSON.stringify({
  'fr': {
    'Hours': 'Heure',
    'Minutes': 'Min'
  }
});
appComponent.use(ui, { language: 'fr' });
appComponent.use(i18n, library);
```

Если создание коллекции возможно только после запуска приложения

```js
created() {
  const library = {
    'Hours': this.gettext('Часы'),
    'Minutes': this.gettext('Минуты'),
  };
  this.$i18n.library = library;
}
```
